# 🚀 Lazy Scripts for Blender (v3.4.1)

**Lazy Scripts** — это динамический фреймворк для Blender, который автоматически превращает ваши Python-скрипты в структурированный графический интерфейс. Больше не нужно вручную регистрировать операторы и классы панелей — просто создайте папку, положите туда скрипт, и интерфейс в N-панели соберется сам.\n\n---\n\n## ✨ Основные возможности\n\n*   **📦 Динамический UI**: Панели и подпанели строятся автоматически на основе файловой структуры ваших папок.\n*   **🔢 Умные переменные**: Встроенная поддержка типов `input_float`, `input_int`, `text` и гибридного `input_custom` (авто-округление, если значение > 1.0).\n*   **🔗 Связь с заголовками**: Используйте `{prop_id}` в названии папки, чтобы выводить значение переменной прямо в шапку панели: `Tools (1.50)`.\n*   **📏 Группировка кнопок**: Файлы с одинаковым цифровым префиксом (например, `10_` и `11_`) автоматически выстраиваются в один горизонтальный ряд (до 3-х элементов).\n*   **⌨️ Горячие клавиши**: Назначайте шорткаты (например, `Ctrl Shift A`) прямо внутри скрипта через словарь `info`.\n*   **⚡ Безопасный запуск**: Скрипты выполняются через `exec()` с доступом к `bpy`, `context`, `report` и прокси-объекту `props`.\n\n---\n\n## 🛠 Архитектура и структура папок\n\nАддон сканирует свою рабочую директорию и интерпретирует файлы следующим образом:\n\n1.  **Папки 1-го уровня**: Становятся главными панелями во вкладке \"Lazy Scripts\".\n2.  **Вложенные папки**: Становятся сворачиваемыми подпанелями (Sub-panels).\n3.  **Именование**: Цифровые префиксы (`00_`, `10_`) и расширения (`.py`) автоматически скрываются в интерфейсе.\n\n### Пример структуры:\n```text\nscripts/\n├── 00_Modeling {my_scale}/\n│   ├── 10_Primitives/\n│   │   ├── 00_Cube.py\n│   │   └── 00_Sphere.py\n│   └── 20_Apply_All.py\n```\n\n---\n\n## 🐍 Создание своего скрипта\n\nКаждый `.py` файл должен содержать словарь `info` в начале. Это конфигурация вашей кнопки или поля ввода.\n\n```python\ninfo = {\n    \"name\": \"Super Tool\",\n    \"type\": \"button\",\n    \"prop_id\": \"my_var\",\n    \"default\": 1.0,\n    \"min\": 0.0, \"max\": 10.0,\n    \"shortcut\": \"Ctrl Shift A\",\n    \"description\": \"Подсказка при наведении\"\n}\n```\n\n---\n\n## ⚙️ Технические особенности (v3.4.1)\n\n*   **Чистый интерфейс**: Технические заголовки классов (`bl_label`) скрыты. Текст в шапках отрисовывается динамически.\n*   **Валидация данных**: Вся логика (Clamp, Round) вынесена в `update_step_logic`.\n*   **Blender 4.5+**: Исправлены ошибки области видимости переменных (`UnboundLocalError`).\n\n---\n\n## 📥 Установка\n\n1. Скачайте репозиторий как ZIP-архив.\n2. В Blender: **Edit > Preferences > Add-ons > Install...**\n3. Выберите ZIP и активируйте **Lazy Scripts**.\n4. Используйте кнопку **Refresh** для обновления списка скриптов.\n\n---\n\n## ⚠️ Требования\n*   **Blender**: 4.0 или выше.\n*   **Python**: 3.10+.\n\n**Разработано для художников, которые ценят свое время.**
