# 🚀 Lazy Scripts for Blender (v3.4.1)

**Lazy Scripts** — это мощный фреймворк для автоматизации, который превращает ваши разрозненные Python-скрипты в профессиональный интерактивный интерфейс. Больше не нужно регистрировать каждый скрипт как отдельный аддон — просто закиньте `.py` файл в папку, и панель управления соберется сама.

---

## ✨ Ключевые особенности

*   **📦 Динамический UI**: Панели, подпанели и кнопки строятся автоматически на основе вашей структуры папок.
*   **🔢 Умные инпуты**: Поддержка типов `input_float`, `input_int` и гибридного `input_custom` (авто-округление до целого, если значение > 1.0).
*   **🔗 Связь с заголовками**: Добавьте `{prop_id}` в название папки, и значение переменной будет отображаться прямо в шапке панели: `My Tool (1.50)`.
*   **📏 Умная группировка**: Файлы с одинаковым цифровым префиксом (например, `10_` и `11_`) автоматически выстраиваются в один горизонтальный ряд (до 3-х элементов).
*   **⌨️ Горячие клавиши**: Назначайте шорткаты (например, `Ctrl Shift A`) прямо внутри скрипта через словарь `info`.
*   **⚡️ Безопасное выполнение**: Скрипты запускаются через `exec()` с доступом к `bpy`, `context`, `report` и прокси-объекту `props` для работы с переменными.

---

## 🛠 Как это работает

Аддон сканирует свою директорию и превращает структуру папок в иерархию панелей Blender (N-panel -> Lazy Scripts).

### Пример структуры папок:
```text
scripts/
├── 00_Modeling/               # Главная панель "Modeling"
│   ├── 10_SubTool {my_var}/    # Подпанель с выводом значения переменной
│   │   └── script.py           # Кнопка внутри подпанели
│   └── 20_Array.py             # Кнопка в основной панели

info = {
    "name": "Super Tool",
    "type": "button",        # Типы: button, input_float, input_int, input_custom, text
    "prop_id": "my_var",     # ID для связи с переменной (опционально)
    "default": 1.0,          # Значение по умолчанию
    "min": 0.0, "max": 10.0, # Лимиты (обрабатываются автоматически)
    "shortcut": "Ctrl Shift A", 
    "description": "Подсказка при наведении"
}

# Ваш код...
# Доступ к переменным: val = props.my_var
# Вывод сообщений: report({'INFO'}, "Done!")
