# 🚀 Lazy Scripts for Blender (v3.4.1)\n\n**Lazy Scripts** — это динамический фреймворк для Blender, который автоматически превращает ваши Python-скрипты в структурированный графический интерфейс. Больше не нужно вручную регистрировать операторы и классы панелей — просто создайте папку, положите туда скрипт, и интерфейс в N-панели соберется сам.\n\n---\n\n## ✨ Основные возможности\n\n*   **📦 Динамический UI**: Панели и подпанели строятся автоматически на основе файловой структуры ваших папок.\n*   **🔢 Умные переменные**: Встроенная поддержка типов `input_float`, `input_int`, `text` и гибридного `input_custom` (авто-округление, если значение > 1.0).\n*   **🔗 Связь с заголовками**: Используйте `{prop_id}` в названии папки, чтобы выводить значение переменной прямо в шапку панели: `Tools (1.50)`.\n*   **📏 Группировка кнопок**: Файлы с одинаковым цифровым префиксом (например, `10_` и `11_`) автоматически выстраиваются в один горизонтальный ряд (до 3-х элементов).\n*   **⌨️ Горячие клавиши**: Назначайте шорткаты (например, `Ctrl Shift A`) прямо внутри скрипта через словарь `info`.\n*   **⚡ Безопасный запуск**: Скрипты выполняются через `exec()` с доступом к `bpy`, `context`, `report` и прокси-объекту `props`.\n\n---\n\n## 🛠 Архитектура и структура папок\n\nАддон сканирует свою рабочую директорию и интерпретирует файлы следующим образом:\n\n1.  **Папки 1-го уровня**: Становятся главными панелями во вкладке \"Lazy Scripts\".\n2.  **Вложенные папки**: Становятся сворачиваемыми подпанелями (Sub-panels).\n3.  **Именование**: Цифровые префиксы (`00_`, `10_`) и расширения (`.py`) автоматически скрываются в интерфейсе.\n\n### Пример структуры:\n```text\nscripts/\n├── 00_Modeling {my_scale}/    # Главная панель с выводом значения\n│   ├── 10_Primitives/         # Подпанель (Sub-panel)\n│   │   ├── 00_Cube.py         # Кнопка 1\n│   │   └── 00_Sphere.py       # Кнопка 2 (в одном ряду с Кнопкой 1)\n│   └── 20_Apply_All.py        # Одиночная кнопка ниже\n```\n\n---\n\n## 🐍 Создание своего скрипта\n\nКаждый `.py` файл должен содержать словарь `info` в начале. Это конфигурация вашей кнопки или поля ввода.\n\n```python\ninfo = {\n    \"name\": \"Super Tool\",\n    \"type\": \"button\",        # Типы: button, input_float, input_int, input_custom, text\n    \"prop_id\": \"my_var\",     # ID для связи с переменной (опционально)\n    \"default\": 1.0,          # Значение по умолчанию\n    \"min\": 0.0, \"max\": 10.0, # Лимиты (обрабатываются автоматически)\n    \"shortcut\": \"Ctrl Shift A\", \n    \"description\": \"Подсказка при наведении\"\n}\n\n# Ваш код здесь...\n# Доступ к переменным: val = props.my_var\n# Вывод в статус-бар: report({'INFO'}, \"Скрипт выполнен!\")\n```\n\n---\n\n## ⚙️ Технические особенности (v3.4.1)\n\n*   **Чистый интерфейс**: Технические заголовки классов (`bl_label`) скрыты. Весь текст в шапках отрисовывается динамически, что исключает дублирование названий.\n*   **Валидация данных**: Вся логика (Clamp, Round) вынесена в `update_step_logic`. В методах отрисовки `draw` разрешено только чтение данных.\n*   **Blender 4.5+ Compatibility**: Исправлены ошибки области видимости переменных (`UnboundLocalError`) при динамической регистрации классов в Python 3.11+.\n*   **Геометрия и Экспорт**: Поддержка Legacy-логики (KD-Tree для поиска дубликатов вершин) и автоматизированный экспорт FBX с кастомными параметрами Scale.\n\n---\n\n## 📥 Установка\n\n1.  Скачайте репозиторий как ZIP-архив.\n2.  В Blender: **Edit > Preferences > Add-ons > Install...**\n3.  Выберите ZIP и активируйте **Lazy Scripts**.\n4.  Используйте кнопку **Refresh** в панели настроек для обновления структуры после добавления новых файлов.\n\n---\n\n## ⚠️ Требования\n*   **Blender**: 4.0 или выше (полная поддержка 4.5+).\n*   **Python**: 3.10+.\n\n**Разработано для художников, которые ценят свое время.**
