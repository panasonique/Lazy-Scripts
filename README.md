# 🚀 Lazy Scripts for Blender (v3.4.1)

**Lazy Scripts** — это динамический фреймворк для Blender, который автоматически превращает ваши Python-скрипты в структурированный графический интерфейс. Больше не нужно вручную регистрировать операторы и классы панелей — просто создайте папку, положите туда скрипт, и интерфейс в N-панели соберется сам.

---

## ✨ Основные возможности

*   **📦 Динамический UI**: Панели и подпанели строятся автоматически на основе файловой структуры ваших папок.
*   **🔢 Умные переменные**: Встроенная поддержка типов `input_float`, `input_int`, `text` и гибридного `input_custom` (авто-округление, если значение > 1.0).
*   **🔗 Связь с заголовками**: Используйте `{prop_id}` в названии папки, чтобы выводить значение переменной прямо в шапку панели: `Tools (1.50)`.
*   **📏 Группировка кнопок**: Файлы с одинаковым цифровым префиксом (например, `10_` и `11_`) автоматически выстраиваются в один горизонтальный ряд (до 3-х элементов).
*   **⌨️ Горячие клавиши**: Назначайте шорткаты (например, `Ctrl Shift A`) прямо внутри скрипта через словарь `info`.
*   **⚡ Безопасный запуск**: Скрипты выполняются через `exec()` с доступом к `bpy`, `context`, `report` и прокси-объекту `props`.

---\n\n## 🛠 Архитектура и структура папок

Аддон сканирует свою рабочую директорию и интерпретирует файлы следующим образом:

1.  **Папки 1-го уровня**: Становятся главными панелями во вкладке \"Lazy Scripts\".
2.  **Вложенные папки**: Становятся сворачиваемыми подпанелями (Sub-panels).
3.  **Именование**: Цифровые префиксы (`00_`, `10_`) и расширения (`.py`) автоматически скрываются в интерфейсе.

### Пример структуры:
```text\nscripts/
├── 00_Modeling {my_scale}/    # Главная панель с выводом значения
│   ├── 10_Primitives/         # Подпанель (Sub-panel)
│   │   ├── 00_Cube.py         # Кнопка 1
│   │   └── 00_Sphere.py       # Кнопка 2 (в одном ряду с Кнопкой 1)
│   └── 20_Apply_All.py        # Одиночная кнопка ниже
```

## 🐍 Создание своего скрипта

Каждый `.py` файл должен содержать словарь `info` в начале. Это конфигурация вашей кнопки или поля ввода.

info = {
    "name": "Super Tool",
    "type": "button",        # Типы: button, input_float, input_int, input_custom, text
    "prop_id": "my_var",     # ID для связи с переменной (опционально)
    "default": 1.0,          # Значение по умолчанию
    "min": 0.0, "max": 10.0, # Лимиты (обрабатываются автоматически)
    "shortcut": "Ctrl Shift A", 
    "description": "Подсказка при наведении"
}

# Ваш код здесь...
# Доступ к переменным: val = props.my_var
# Вывод в статус-бар: report({'INFO'}, "Скрипт выполнен!")

## ⚙️ Технические особенности (v3.4.1)

*   **Чистый интерфейс**: Технические заголовки классов (`bl_label`) скрыты. Весь текст в шапках отрисовывается динамически, что исключает дублирование названий.
*   **Валидация данных**: Вся логика (Clamp, Round) вынесена в `update_step_logic`. В методах отрисовки `draw` разрешено только чтение данных.
*   **Blender 4.5+ Compatibility**: Исправлены ошибки области видимости переменных (`UnboundLocalError`) при динамической регистрации классов в Python 3.11+.
*   **Геометрия и Экспорт**: Поддержка Legacy-логики (KD-Tree для поиска дубликатов вершин) и автоматизированный экспорт FBX с кастомными параметрами Scale.

## 📥 Установка

1.  Скачайте репозиторий как ZIP-архив.
2.  В Blender: **Edit > Preferences > Add-ons > Install...**
3.  Выберите ZIP и активируйте **Lazy Scripts**.
4.  Используйте кнопку **Refresh** в панели настроек для обновления структуры после добавления новых файлов.
